************************************
splunk.es.splunk_data_inputs_network
************************************

**Manage Splunk Data Inputs of type TCP or UDP resource module**


Version added: 2.1.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Module that allows to add/update or delete of TCP and UDP Data Inputs in Splunk.
